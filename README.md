# 🧱 Brick Breaker

パドルでボールを跳ね返してブロックをすべて壊すブロック崩しゲーム。

## 遊び方
- **マウス移動** または **←→** — パドルを操作
- **クリック / SPACE** — ボール発射
- ブロックを全部壊すとレベルアップ

## パワーアップアイテム
| アイテム | 効果 |
|---|---|
| **W** (Wide) | パドルが大きくなる |
| **M** (Multi) | ボールが3つに増える |
| **S** (Slow) | ボールがスローになる |

## 特徴
- レベルが上がるほどブロックの耐久度が増す
- ボールの軌跡エフェクト
- 3ライフ制
- パワーアップシステム

## 技術スタック
HTML5 Canvas / Vanilla JavaScript
